# io-react-native-cie-pid

A RN package for pid issuing based on the IPZS native SDK

## Installation

```sh
npm install io-react-native-cie-pid
```

## Usage

```js
import { multiply } from 'io-react-native-cie-pid';

// ...

const result = await multiply(3, 7);
```
